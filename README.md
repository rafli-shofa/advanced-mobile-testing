# Advanced Mobile Testing with Appium

Project ini merupakan tugas **INDIVIDU** Advanced Mobile Automation Testing menggunakan **Appium + WebdriverIO + TypeScript**.

Automation dilakukan pada aplikasi **ApiDemos-debug.apk**, khususnya fitur **Text Entry Dialog**, dengan tujuan memverifikasi input field *name* dan *password* sesuai dengan data yang diinput.
